# GridVox Livery Builder - Concept & Vision

**Project Name:** GridVox Livery Builder (working title: "PaintVox" or "DesignVox")  
**Tagline:** "Your story. Your style. Your sim."  
**Date:** November 12, 2025  
**Vision:** A community-driven, AI-powered, multi-sim livery designer deeply integrated with the GridVox ecosystem

---

## Executive Summary

GridVox Livery Builder is a revolutionary browser-based and desktop livery design tool that transcends traditional paint software by integrating AI assistance, social collaboration, persona-driven creativity, and story-based design challenges. Unlike Trading Paints (iRacing-only) or standalone tools (Photoshop, GIMP), GridVox Livery Builder:

1. **Supports multiple sims** (iRacing, ACC, AMS2, rFactor 2, Forza, Gran Turismo, F1, WRC, Wreckfest)
2. **Leverages AI** for design assistance, color suggestions, logo generation, and voice commands
3. **Integrates with GridVox personas** for creative feedback and coaching
4. **Ties to GridVox stories** enabling story-driven livery challenges and progression
5. **Is community-powered** with a credit system rewarding contributors
6. **Offers real-time collaboration** for team projects
7. **Provides AR preview** to see liveries on physical objects
8. **Enables voice control** using GridVox's voice command system

### Template Conversion System

**Problem:** Each sim has different:
- Texture layouts (UVs)
- File formats (.tga, .dds, .png)
- Resolution requirements
- Material system (spec maps, normal maps)
- Car naming conventions

**Solution:** GridVox Livery Builder's Template Converter

**How It Works:**
1. **Design Once:** Create livery on universal canvas
2. **Select Car:** Choose car from any sim
3. **Auto-Map:** AI maps design to that car's UV layout
4. **Export:** Generate sim-specific files automatically
5. **Multi-Export:** Export same design to multiple sims with one click

**Example:**
- Design a GT3 livery for McLaren 720S
- Export to:
  - iRacing: McLaren 720S GT3 (.tga + spec maps)
  - ACC: McLaren 720S GT3 Evo (.dds layered)
  - AMS2: McLaren 720S GT3 (.dds)
  - rFactor 2: McLaren 720S GT3 (.dds + spec)
- **One design, four sims, one click**

### Car Database

**Required Data:**
- Sim name
- Car name (internal and display)
- Template file (blank UV layout)
- Texture resolution
- Supported maps (diffuse, spec, normal, emissive)
- File format
- Installation path (optional, for auto-install)

**Community Contribution:**
- Users upload templates for cars
- Earn credits for templates used by others
- Validation by moderation team
- Automatic template generation (AI traces 3D model)

## Technical Architecture (High-Level)

### Frontend Stack
- **Framework:** React + TypeScript
- **3D Rendering:** Three.js for WebGL 3D preview
- **Canvas Rendering:** Fabric.js or Konva.js for 2D layer editing
- **State Management:** Zustand or Redux Toolkit
- **UI Components:** Custom component library + Radix UI primitives
- **Styling:** Tailwind CSS + CSS Modules
- **Animation:** Framer Motion
- **Desktop App:** Tauri (Rust backend, web frontend)
- **Mobile:** React Native with shared components

### Backend Stack
- **API:** Node.js + Express or Fastify
- **Database:** PostgreSQL (relational data) + Redis (caching)
- **File Storage:** AWS S3 or Cloudflare R2 (assets, projects)
- **Authentication:** Supabase Auth or Auth0
- **Real-Time:** WebSockets (Socket.io or Pusher)
- **AI Services:** OpenAI API, Replicate, or self-hosted Stable Diffusion

### AI Integration
- **Local LLM:** (Optional) Lightweight model for voice commands
- **Cloud LLM:** OpenAI GPT-4 or Anthropic Claude for complex requests
- **Image Generation:** Stable Diffusion XL for logos, patterns
- **Voice Recognition:** Whisper API or Web Speech API
- **Voice Synthesis:** ElevenLabs or Azure TTS (for persona voices)

### GridVox Integration APIs
- **Persona API:** Fetch persona data, trigger voice responses
- **Story API:** Get story requirements, update livery progression
- **Social API:** Post to social media, fetch rivalries
- **Voice API:** Send voice commands, receive transcriptions
- **Credit API:** Manage credit balance, transactions

## User Workflows

### Workflow 1: New User - First Livery (Beginner Mode)

1. **Sign Up:** Email or social login
2. **Welcome Tutorial:** Interactive 5-minute tour
3. **Choose Starting Point:**
   - Blank canvas (for advanced users)
   - Template library (recommended)
   - Clone community livery
4. **Select Template:** Browse categories, pick "GT3 Modern Racing"
5. **Customize:**
   - Change team colors (AI suggests harmonious palettes)
   - Upload sponsor logos (auto-background removal)
   - Add racing number (choose from number styles)
   - Adjust placement (smart guides snap logos into place)
6. **Preview:** Rotate 3D model to see all angles
7. **AI Feedback:** "Darth Vader thinks this needs more contrast. Make the logos pop!"
8. **Export:** Select sim (iRacing), choose car (Porsche 911 GT3 R)
9. **Upload:** Direct to Trading Paints (iRacing) or download files
10. **Share:** Post to social media, share with friends

**Time:** 15-30 minutes for first livery

### Workflow 2: Experienced User - Custom Livery from Scratch

1. **New Project:** Blank canvas
2. **Select Car:** Choose from multi-sim car database
3. **Base Paint:** Select iRacing base pattern or solid color
4. **Draw Shapes:** Use pen tool to draw custom racing stripes
5. **Add Gradients:** Apply gradient fill with custom colors
6. **Import Logos:** Drag-and-drop sponsor logos from computer
7. **Vector Graphics:** Add racing numbers from library
8. **Apply Effects:** Drop shadows, outlines, blend modes
9. **Material Finishes:** Chrome logos, matte base, metallic accents
10. **Sim Preview:** Launch iRacing viewer, see in real lighting
11. **Iterate:** Make adjustments based on preview
12. **Voice Command:** "Export this to iRacing, ACC, and AMS2"
13. **Multi-Sim Export:** One click, three sim formats
14. **Version Save:** Name version "Final - Championship 2025"
15. **Upload to Community:** Share to showroom, earn credits

**Time:** 1-3 hours for detailed custom livery

### Workflow 3: Team Collaboration - Multiplayer Story Livery

1. **Story Selection:** GridVox story mode "1974 F1 Championship"
2. **Team Formation:** Invite 3 friends to race together
3. **Livery Requirements:** Story requires 1970s-accurate liveries
4. **Team Template:** Start from shared team base (Lotus 72)
5. **Assign Roles:** Designer 1 (base colors), Designer 2 (sponsors), Designer 3 (numbers)
6. **Real-Time Collab:** All edit simultaneously, see each other's cursors
7. **Chat:** In-app chat to discuss changes
8. **AI Validation:** "This color wasn't available in 1974. Suggest period-accurate alternatives?"
9. **Persona Feedback:** Jackie Stewart (historical persona): "That Gold Leaf livery is spot on!"
10. **Version Approval:** Team votes to approve final design
11. **Batch Export:** Export for all 4 team members (different numbers)
12. **Story Integration:** Liveries automatically sync to GridVox story mode
13. **Race:** Use liveries in story championship races

**Time:** 2-4 hours collaborative design session

### Workflow 4: Voice-Controlled Quick Edit (Hands-Free)

**Scenario:** User is racing in a few hours, wants to quickly update sponsor for new sponsor deal.

1. **Voice:** "Hey GridVox, open my Championship Livery"
2. **AI:** "Opening Championship Livery. What would you like to change?"
3. **Voice:** "Replace the front bumper sponsor with the new RedBull logo"
4. **AI:** "Selecting front bumper sponsor. Searching graphics library for RedBull logo. Found it. Replacing now."
5. **Voice:** "Make it 20% bigger"
6. **AI:** "Scaling logo to 120%. Done."
7. **Voice:** "Show me the iRacing preview"
8. **AI:** "Launching iRacing viewer. One moment."
9. **Voice:** "Looks good. Export to iRacing and upload to Trading Paints."
10. **AI:** "Exporting to iRacing format... Uploading to Trading Paints... Done. Your livery is live."

**Time:** 2-5 minutes
