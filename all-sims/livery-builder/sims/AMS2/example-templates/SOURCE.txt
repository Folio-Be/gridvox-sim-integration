## Template Format

- **Container:** RAR compressed archives
- **Content:** Photoshop PSD files (layered)
- **Layers Typically Include:**
  - Base paint layer (UV-mapped)
  - Wireframe overlay (for panel identification)
  - Ambient occlusion (AO) layer
  - Decal/sponsor placement guides
  - Window/glass masks
  - Number plate regions

## Export Requirements

Per AMS2 custom livery system:
- **Format:** DDS (DirectDraw Surface)
- **Texture Names:** Match vehicle-specific naming in `Overrides/{vehicle}/_dist.xml`
- **Installation Path:** `C:\GAMES\Automobilista 2\Vehicles\Textures\CustomLiveries\Overrides\`
- **Multiplayer:** Local only (custom liveries not visible to other players)

## Official Documentation

**Reiza Custom Livery System Guide:**
- Navigate to `Automobilista 2\Vehicles\Textures\CustomLiveries\Overrides\`
- Each car has `_dist.xml` with installation instructions
- XML specifies texture file names, livery slots, helmet overrides
