# Livery Template Sources

The folders below are staged to hold up to five vetted template examples per simulator. Due to licensing and distribution restrictions, the actual template packages must be downloaded from the official or community portals listed here. Each entry includes the access requirements and the most recent public update indicator available as of 2025-11-12.

## Automobilista 2 (AMS2)
- Reiza Studios Forum – Automobilista 2 Custom Livery Overview *(free registration required)*  
  https://forum.reizastudios.com/threads/automobilista-2-livery-customization-overview.9745/
- RaceDepartment – Automobilista 2 Template Megapack *(free account required, shows update timestamp)*  
  https://www.racedepartment.com/downloads/automobilista-2-paint-template-megapack.38852/
- Google Drive Bundle (linked inside the forum thread) *(latest official pack; check file modified dates)*
- JustRace Community Pack *(free account)*  
  https://www.justrace.net/downloads/category/automobilista-2/
- Racing Club Discord *(request access to #ams2-resources for bespoke templates)*  
  https://discord.gg/racingclub
